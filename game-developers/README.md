# Manual for GZRO Graviteers

## About GZRO.

GZRO is a decentralized, blockchain based cryptocurrency that offers fast and secure transactions. By taking advantage of the tools and resources offered by the GZRO Game Market, both players and the game developers will be able to effectively and conveniently be a part of – and be involved in - everything the online gaming world has to offer.

We are looking for graviteers to add more examples in different languages. Please feel free to submit a pull request if you have any code that might help others integrating GZRO in their games.

## Welcome to our community.
Our community is open for all game developers who use GZRO in their software. Please join the community on Telegram, Discord or Twitter. The graviteers community welcomes you and is commited to delivering you a great experience with mind-blowing events, fun and the latest tech at your fingertips!
